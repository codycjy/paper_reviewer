# Review

## Summary
This paper proposes a method for unsupervised pose estimation, namely Pose Prior Learner (PPL). The key idea is to learn a general pose prior for an object category from images in a self-supervised manner. The authors claim that the proposed method outperforms existing methods across several pose estimation benchmarks.
