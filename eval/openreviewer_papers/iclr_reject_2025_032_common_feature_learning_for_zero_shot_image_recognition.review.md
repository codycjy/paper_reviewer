# Review

## Summary
This paper introduces the Common Feature learning for Zero-shot Image Recognition (CF-ZIR) method, which establishes fine-grained visual-semantic relationships at the image level. By leveraging inter-class association information from class semantic vectors, CF-ZIR guides the extraction of common visual features, resulting in more nuanced image semantic vectors.

## Strengths
1. The paper is well-organized and easy to understand.
2. The proposed method achieves good performance on several datasets.

## Weaknesses
1. The paper is not well-written and contains numerous typos, such as "inter class association information provided by class semantic vectors guide the extraction" in the abstract, and "totally, the loss function of the visual-semantic coupled dictionary learning stage is as follows" on page 5.
2. The proposed method lacks innovation, as it merely combines attribute information with visual information for learning, which is a common approach in zero-shot learning.
3. The paper lacks experiments on the CUB dataset and comparisons with recent methods, such as "Attribute-Guided Graph Neural Network for Zero-Shot Learning" (NeurIPS 2022), "Attribute-Guided Transformer with Multi-Level Adaptation for Zero-Shot Learning" (TIP 2023), and "Attribute-Guided Meta-Learned Neural Network for Zero-Shot Learning" (PR 2024).
